## [PT-BR] (English below)

Este é um projeto simples de um robô (bot) do Discord, que eu utilizo até hoje com meus amigos. Feito utilizando Python, ele é capaz de, a pedidos de usuários, juntar-se a canais de voz e -- por meio da API do Youtube -- procurar músicas com base no input dos usuários e tocá-las no canal de voz. O usuário também é capaz de pular músicas (skip) e parar (stop), assim como expulsar o bot do canal de voz (leave).

Atualmente, eu deixo ele rodando em meu computador pessoal quando quero usá-lo.

## [EN]

This is a simple working project of a Discord bot, which I still use with my friends. Built using Python, it's capable of joining voice channels and -- through Youtube's API -- search for songs or videos based on the user's input and play those songs in the voice channel. The user is also able to skip songs (skip), stop (stop), as well as forcing the bot to leave).

I currently get it running on my personal computer when I want to use it.
